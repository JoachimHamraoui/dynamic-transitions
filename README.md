## Run Locally

Clone the project

```bash
  git clone https://github.com/JoachimHamraoui/dynamic-transitions.git

Go to the project directory

```bash
  cd dynamic-transitions/transitions
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Tech Stack

**Client:** React, Vite, TailwindCSS
